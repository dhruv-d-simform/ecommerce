# Ecommerce

## Overview

This project is a simple e-commerce application with the following features:

### Header

- A sticky header containing:
    - A fake logo.
    - A search bar to search dummy products.

### Sidebar

- A non-functional sidebar on the left displaying static filters.

### Main Content Area

- To the right of the sidebar, a list of available products is displayed.
- Each product card includes:
    - Photo
    - Title
    - Description
    - Price
    - Discount
    - Calculated price after discount
    - Availability status

### Product Details View

- Clicking on a product card hides the sidebar and displays the product details in the main content area.
- The header and footer remain visible while viewing product details.
- Product details include:
    - A cross (x) button at the top-right corner to close the product details view.
    - Clicking the cross button restores the product list and sidebar.

### Footer

- A footer containing:
    - A fake copyright disclaimer.
    - Dummy contact information and address.

## Features

- Sticky header for seamless navigation.
- Sidebar for static filters.
- Dynamic product list with detailed information.
- Toggle between product list and individual product details.
- Persistent header and footer for consistent user experience.
